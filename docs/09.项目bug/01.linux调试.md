# 编译链接了对相路径，但运行时没有动态库

当在链接时指定-L选项，是告诉链接器去哪找到库。但是运行时操作系统需要知道在哪里找到需要的共库

export LD_LIBRARY_PATH=$LD_LIBRARY_PATH:/usr/local/lib

将目标库的路径注册进环境变量，
